Project Background

We built this project for our Software Engineering class where we were tasked to either change or add a component to a Learning Management System.

We chose to change and build upon a “Quick Links” system on BlackBoard for UMass Lowell. The current “Quick Links” system is available directly beneath the UML logo in the top left of the BlackBoard home page. It can be accessed by hovering over the chain logo. A lot of links can be clicked on however, none of them change the current page. The system is essentially useless as it is so, we decided to make that small component into something that could be usable.

Our project allows users quick access to several different parts of BlackBoard such as assignments, resource folders, class recordings, etc. They gain this quick access feature through our bookmark system. It allows the user to choose which parts of the BlackBoard they get quick access to. It saves time because instead of going through different folders every time they can click on one button and go straight to what they want.
